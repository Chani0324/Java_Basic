JPA 학습

1. DB연동 표준 API
	- jdbc보다 개발 코드를 더 추상화 해놓은 db연동 Framework
	
2. 추상화란?
	- 코드가 간소화
	- 설정만으로도 sql 문장 자동 생성
	- 메소드 호출만으로 데이터 crud
	... 등의 하드 코딩 작업 없이도 로직 개발이 가능
	- 고급기술이 될 수록 추상화
	
3. 작업단계
	1. 개발환경 세팅
		jdk 설치
		ide(통합개발 tool) 설치
		db 설치
	2. 개발
		jpa 개발이 가능한 project 생성
		개발 환경에 최적으로 호환되는 framework(library)들 설정
